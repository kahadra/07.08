고정 배경이 존재하지만, 일반 detector는 현재 프레임 한 장만 본다.
  • 정지 카메라에서는 배경이 거의 고정되어 있음
  • 현재 프레임과 배경의 차이는 객체 검출에 직접적인 단서가 될 수 있음
  • 일반 detector는 이 차이를 별도 입력이나 구조로 반영하지 않음
  • 결과적으로 환경적 이점을 버리고 일반 이미지 검출 문제처럼 처리함
일반 YOLO 계열 detector는 현재 프레임만 보고 객체를 검출합니다. 하지만 정지 카메라에서는 배경이 고정되어 있기 때문에 현재 이미지와 배경의 차이 자체가 중요한 단서가 될 수 있습니다.

Main Idea 06 / 18

Background Image-Assisted Detection

Figure 2. Image-level fusion framework for BGI-YOLO
Foreground mask를 직접 만들지 않고, 배경 이미지를 detector 입력으로 준다.
  • 현재 이미지와 background image를 함께 입력
  • YOLO가 두 이미지의 차이를 end-to-end로 학습
  • 배경 정보를 후처리 규칙이 아니라 detector 내부 정보로 사용
  • hard mask 생성에 따른 binarization 실패를 피하려는 접근

Feature-Level Fusion

Figure 3. Feature-level fusion framework
두 이미지를 각각 backbone에 통과시킨 뒤 P3, P4, P5 feature map을 결합한다.

BGI-YOLO(2): Concatenate

같은 level의 feature map을 channel 방향으로 결합한다.

BGI-YOLO(3): Add

같은 level의 feature map을 element-wise addition으로 결합한다.

Cost: Dual Backbone

두 backbone을 사용하므로 연산량이 크게 늘지만, 성능은 BGI-YOLO(1)보다 낮았다.

BGI-YOLO(2)와 (3)은 입력 단계가 아니라 feature 단계에서 결합합니다. 하지만 두 이미지를 각각 backbone에 통과시키기 때문에 연산량이 크게 증가하고, 결과적으로 image-level fusion보다 좋지 않았습니다.

Discussion 17 / 18

Strengths and Weaknesses

Strengths

  • 정지 카메라라는 환경 조건을 명확히 활용함
  • image-level fusion 구조가 단순함
  • 연산량 증가가 거의 없음
  • background subtraction의 hard mask 문제를 피함
  • mAP, precision, recall 모두 개선 확인

Weaknesses

  • stationary camera assumption이 강함
  • background image 품질에 의존함
  • 장기적 배경 변화 대응 분석은 제한적
  • 최신 YOLO 계열이나 transformer detector 비교가 부족함
  • latency, false alarm, 장기 운용 안정성 분석은 제한적
이 논문의 장점은 정지 카메라 조건을 잘 활용하고, 특히 image-level fusion은 단순하면서 연산 증가가 거의 없다는 점입니다. 다만 안정적인 배경 이미지와 고정 카메라에 강하게 의존하며 장기 운용 분석은 충분하지 않습니다.
